Ubuntu16.04下Aeternity测试链挖矿

Posted on 2018-03-03 | In BlockChain

全文参考链接

写在前面(把ubuntu的本地文件用ssh传到云端服务器)

scp 本地文件地址+文件名 远程用户名@IP地址:+服务器内存放文件的地址
EXAMPLE: scp /home/user/aa.txt root@103.14.134.247: /root

安装libsodium

完成之后查看是否存在这个文件/usr/local/lib/libsodium.so.23，如果有则安装成功。

下载release binary文件

到Aeternity的github上下载ubuntu版本文件。
NOTE:我是用的阿里云服务器，所以我在自己电脑上下载好之后又传到服务器上
scp /home/yann/epoch-0.8.0-ubuntu.tar.gz root@47.100.47.252:/root
以上是将本地文件传到云服务器

部署节点

NOTE: 以上为创建新目录/tmp/node并将/root路径下的tar.gz文件解压到/tmp/node路径下

配置节点

创建配置文件vi epoch.yaml,将以下内容添加进文件，同时用自己的地址和端口分别修改http > external > peer_address和http > external > port下的内容。

完成之后保存并退出，bin/epoch check_config epoch.yaml验证配置文件，若配置成功则可以返回OK。

启动节点

bin/epoch start启动节点 如果有需要可以bin/epoch stop关闭

备份私钥

备份私钥：cp -pr /tmp/node/keys /root/my_epoch_keys

验证及信息查看

  • 验证节点是否在链上curl http://127.0.0.1:3003/v2/top

  • 查看挖矿信息：less /tmp/node/log/epoch_mining.log，如果正在挖矿，会显示如下信息：

  • 如果成功挖到块，会显示：... Block mined: Height = 1; Hash = ...

  • 验证你的节点是否连接到测试网，(即高度是否相同)
    查看测试网当前最高区块curl http://31.13.249.1:3013/v2/top
    查看自己节点当前最高区块curl http://127.0.0.1:3003/v2/top
  • 验证你的节点和测试网是否在同一区块上挖矿
    查看远程节点的哈希值和自己节点的前一哈希是否相同。
    curl http://31.13.249.1:3013/v2/top
    curl http://localhost:3103/v2/block/pending
