如何查看请求发送地址

打开登陆页面，然后按F12查看控制台，选择Network选项，这时在网页点击登陆，查看控制台变化，这时候应该name下会新增几条记录，通过查看这几个name记录查看登陆和验证码的请求地址。具体可查看下图：
